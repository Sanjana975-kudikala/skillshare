# 🚀 SkillShare – Project Collaboration Platform  

🔗 **Live Demo:** https://skillshare-indol.vercel.app/  

---

## 🌟 Overview  
**SkillShare** is a full-stack **MERN-based collaboration platform** designed for college students to **connect, collaborate, and build innovative projects**.

It enables users to:
- 🔍 **Discover peers** with similar skills  
- 💡 **Share project ideas**  
- 🤝 **Collaborate on real-world projects**  
- 📁 **Build a strong portfolio**  

---

## ✨ Key Features  

### 📊 **Dashboard**
-  **Create and manage projects**  
-  Track project status (**Idea / Planned / Completed**)  
-  Maintain a **personal portfolio**  

### 🤝 **Network Hub**
- **Search and discover peers**  
-  Send and manage **connection requests**  
-  View **user profiles and portfolios**  

### 🧠 **Skill Management**
-  Add, edit, and remove **technical skills**  
-  Skill-based filtering for better collaboration  

### 🔗 **Professional Profiles**
-  Add **GitHub & LinkedIn links**  
-  Secure access to user information  

### 🎨 **UI/UX**
-  Clean and modern **dashboard design**  
-  Fully **responsive layout**  
-  Smooth and intuitive **user experience**

```bash

cd client

npm start
